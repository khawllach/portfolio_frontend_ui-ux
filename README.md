## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/khawllach/portfolio_frontend_ui-ux.git
```

Navigate to the project:

```bash
cd portfolio_frontend_ui-ux
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
